# 내 비즈니스 파트너 (AI 비서) — 시작하기

텔레그램으로 대화하는, **내 사업을 아는 전담 AI 비서**입니다.
일정 브리핑 · 기록 · 회상 · 초안 · 이슈 관리 · 리마인더를 메시지 한 문장으로 처리합니다.

이 폴더가 그 비서의 "몸"입니다. 아래 순서대로만 하면 됩니다. 개발 지식은 필요 없습니다.

---

## 내려받기

둘 중 편한 방법으로 받으세요:

- **ZIP (권장·가장 쉬움)** — 이 GitHub 페이지에서 초록색 **Code → Download ZIP** 을 눌러 내려받고, 더블클릭으로 압축을 풉니다.
- **git** — 터미널에서: `git clone https://github.com/hminnnnnn/my-biz-partner.git`

## 설치 (터미널 명령 2개)

> 준비물: Mac · 텔레그램 계정 · Claude Code 설치. (자세한 준비물 안내는 워크숍에서 도와드려요.)

**1) 받은 폴더로 이동합니다.**  (압축을 푼 위치에 따라 경로는 바꿔서)

```
cd ~/Downloads/my-biz-partner
```

**2) 설치 스크립트를 실행합니다.**

```
./install.sh
```

끝입니다. 스크립트가 필요한 폴더와 초기 파일을 알아서 만들고, 다음에 뭘 할지 안내해 줍니다.
(이미 설치돼 있으면 기존 기록을 지우지 않고 그대로 둡니다. 여러 번 실행해도 안전합니다.)

---

## 설치 다음에 할 일 — 터미널에서 코치를 부르세요

설치가 끝나면, **이 폴더에서 터미널에 `claude` 를 켜고 이렇게 한마디만 하세요:**

```
셋업 시작
```

그러면 **셋업 코치**가 지금 어디까지 됐는지 조용히 진단한 뒤, **텔레그램 연결 → 정체성 채우기 → 캘린더 → 첫 대화**를 순서대로, 한 번에 하나씩 안내하고 매 단계 진짜 됐는지 확인해 줍니다. 문서를 혼자 따라갈 필요가 없습니다. 세션이 꺼져도 다시 "셋업 시작"이라고 하면 **멈춘 지점부터** 이어서 끝까지 데려갑니다.

> 개발 지식은 필요 없습니다. 막히면 코치가 무엇이 문제인지 짚어 줍니다.

<details>
<summary><b>코치 없이 직접 하고 싶다면 (수동 순서)</b></summary>

1. **텔레그램 연결** — `docs/SETUP_TELEGRAM.md` 를 따라 봇을 만들고 이 컴퓨터와 연결합니다.
2. **나를 소개하기** — `identity/IDENTITY.md` 의 4칸을 채웁니다. (예시는 `identity/EXAMPLE.md`.)
3. **첫 대화** — 텔레그램에 아무 말이나 걸어 보세요:
   - "안녕, 내 사업 알고 있어?"
   - "방금 A업체가 단가 조정 요청했어" (기록됩니다)
   - "내일 일정 브리핑해줘"
4. **캘린더 연결(선택)** — `docs/SETUP_CALENDAR.md`. 연결이 어려우면 폴백 방법도 있습니다.
5. **대시보드 열기** — `dashboard/index.html` 을 더블클릭하면 내 비서 상태가 보입니다.

</details>

> 전체 사용법(기능 사전·셋업 여정·자주 막히는 곳)은 `docs/GUIDE.md` 를 참고하세요.

## 이 폴더 안에 뭐가 있나요

| 위치 | 무엇 |
|---|---|
| `docs/GUIDE.md` | **전체 사용 가이드** — 기능 사전·셋업 여정·자주 막히는 곳 |
| `guide.html` | **인터랙티브 가이드** — 더블클릭해서 여는 사용법 체험 (진행 체크·대화 미리보기) |
| `identity/IDENTITY.md` | **당신이 채우는 유일한 파일** — 사업·고객·말투·우선순위 |
| `CLAUDE.md` | 파트너의 행동 지침 (건드리지 않아도 됩니다) |
| `notes/` | 던진 메모가 분류되어 쌓이는 곳 |
| `dashboard/index.html` | 내 비서 상태를 보는 화면 (더블클릭해서 열기) |
| `docs/` | 텔레그램·캘린더 연결 가이드, 상태 계약 설명 |
| `.claude/skills/` | 기능 모듈 (브리핑·기록·초안 등) — 자동으로 쓰입니다 |

- 막히면 워크숍에서 함께 풀어 갑니다. 완벽하지 않아도 괜찮습니다.
