# draft-writer — 내 말투 초안

## 언제 쓰나
"이 문의에 답장 초안", "공지 초안 써줘", "이 메일 답장", "정중히 거절하는 문구".
응대·공지·메일을 대표 톤으로 빠르게 뽑는 기능. 시간 절약이 즉시 체감되는 순간(W4).

## 동작 순서
1. **말투 로드** — `identity/IDENTITY.md` 의 "3. 내 말투·톤"과 사업 맥락을 먼저 읽는다.
   (미기입이면 초안 전에 톤을 한 번 물어본다: "정중한 존댓말로 갈까요, 친근하게 갈까요?")
2. **맥락 보강** — 특정 고객·업체 대상이면 `notes/` 에서 관련 기록을 찾아(=recall 규약) 반영한다.
3. **초안 작성** — 대표 말투로, 목적(수락/거절/안내/사과)에 맞게. 길이는 용도에 맞게 조절.
4. **확인 요청** — "이대로 보낼까요? 고칠 부분 있으세요?" 로 마무리. 확정·발송은 대표 몫.
5. **초안 보관** — 초안을 대상 고객·업체의 `notes/customers/` 파일에 "## 초안 (날짜)" 섹션으로
   이어 적는다 (해당 파일이 없으면 새로 만들고, 대상이 불특정하면 `notes/inbox/` 에 저장).
   나중에 "그때 그 초안" 을 recall 로 다시 찾을 수 있게 하기 위함이다.
6. **상태 갱신** — `state/status.json` 의 `today.counts.drafts` +1, `today.activities` 에
   `{time,type:"draft",summary}` 추가, `updatedAt`·`lastActiveAt` 갱신 후 `state/dashboard-data.js`
   재생성 (`docs/STATE_CONTRACT.md` 준수).

## 읽는 위치
- `identity/IDENTITY.md` (말투·사업 맥락)
- 필요 시 `notes/` (대상 고객·업체 관련 기록)

## 출력 형식 예시
```
[답장 초안 — 정중한 존댓말]
안녕하세요, 대표님. 문의 주셔서 감사합니다.
말씀하신 단가 건은 내부에서 검토 후 이번 주 안에 회신드리겠습니다.
급하신 부분이 있으면 편하게 말씀해 주세요. 감사합니다.

— 이대로 보낼까요? 톤이나 내용 고칠 곳 있으면 알려주세요.
```
